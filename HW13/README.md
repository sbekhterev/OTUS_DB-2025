## Подготовка

С помощью скрипта [add_products](./script/add_products_mysql.py) заполним базу тестовыми продуктами.
Так же создадим VIEW для удобства написания запросов (будем считать недоступными товары у которых нет цен).
```amplicode sql
CREATE VIEW v_products AS (
SELECT pc.CategoryName, 
       p.ProductName, p.EAN, 
       pu.UnitName, p2.Price, 
       CASE WHEN p2.Price IS NULL THEN 'sold_out' ELSE 'in_stock' END AS Availability
  FROM Products p 
  JOIN ProductsProductCategory ppc ON ppc.ProductsId = p.ID 
  JOIN ProductCategory pc ON pc.ID = ppc.ProductCategoryId
  JOIN ProductUnit pu ON pu.Id = p.ProductUnitId
  LEFT JOIN (
  			SELECT p.ID, p.ProductId, p.Price, pl.EndDate 
  			  FROM Price p 
  			  JOIN PriceList pl on pl.Id = p.PriceListId and pl.EndDate IS NULL) as p2 on p2.ProductId = p.id
);
```

```amplicode sql
SELECT vp.CategoryName,
	   COUNT(
	   CASE 
	   	WHEN vp.UnitName = 'кг' THEN 1 ELSE 0
	   END) AS 'Весовой товар'
  FROM v_products vp  
 GROUP BY vp.CategoryName
 HAVING  
 	   COUNT(
	   CASE 
	   	WHEN vp.UnitName = 'кг' THEN 1 ELSE 0
	   END) > 5500;
```

## Сделать выборку, показывающую самый дорогой и самый дешевый товар в каждой категории (такие названия нагенерировались):
```amplicode sql
WITH cte as (
SELECT vp.CategoryName,
       vp.ProductName,
       vp.Availability,
       vp.Price,
       ROW_NUMBER() OVER (PARTITION BY vp.CategoryName ORDER BY vp.Price DESC) as Max_Price,
       ROW_NUMBER() OVER (PARTITION BY vp.CategoryName ORDER BY vp.Price ASC) as Min_Price
  FROM v_products vp
 WHERE vp.Availability = 'in_stock')
SELECT c.CategoryName,
       MAX(CASE WHEN c.Max_Price = 1 THEN c.ProductName END) as Max_Product,
       MAX(CASE WHEN c.Max_Price = 1 THEN c.Price END) as Max_Price,
       MAX(CASE WHEN c.Min_Price = 1 THEN c.ProductName END) as Min_Product,
       MAX(CASE WHEN c.Min_Price = 1 THEN c.Price END) as Min_Price
  FROM cte c
  GROUP BY c.CategoryName
  ORDER BY c.CategoryName;
```

```amplicode sql
  FROM v_products vp
  GROUP BY vp.CategoryName WITH ROLLUP;
```
